Лаунчер для Counter-Strike 2/Go.

Ето оссобеность в том, что он может в один клик устанавливать старые или лимитные версии КС2 без установки всего контента. Например:
*Csgo 2013
*csgo legacy (appID:730)
*Cs2 beta
*Cs2.
Вскоре будет больше.

Основан на DepotDownloader, ссылка:https://github.com/steamre/depotdownloader

Плюсы:
• работа на Windows 7
• установка отдельного контента без установки всей игры
• удобный и понятный интерфей
